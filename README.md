Capconnect -- Kotlin Dating App
 Project Description
**Capconnect** is a mobile dating app developed in
Kotlin with Jetpack Compose.
This project aims to create a simple and intuitive platform
allowing users to discover new people, like
profiles, and chat with their matches.
The project also serves as a learning platform for modern
Android programming, state management with ViewModel, and interface design with Compose.
